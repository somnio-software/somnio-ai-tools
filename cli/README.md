# Somnio CLI

The Dart CLI that installs AI-powered audit skills and executes multi-step audits.

For full documentation, see the [CLI Reference](../docs/cli.md).

## Quick Start

```bash
dart pub global activate -sgit https://github.com/somnio-software/somnio-ai-tools.git --git-path cli
somnio setup
somnio run fh
```

## Development

```bash
dart pub get       # Install dependencies
dart analyze       # Run static analysis
dart test          # Run all tests
```

For the full development guide, see [Contributing](../docs/contributing.md).
